Stabilize a reverse shell

After getting a reverse shell, we can stabilize it using commands below.

  • python3 -c 'import pty;pty.spawn("/bin/bash")'
  • export TERM=xterm
  • ctrl+z
  • stty raw -echo; fg
  • stty rows 38 columns 116
